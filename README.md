 - O Que são Classes e Objetos em C#
   - Classe: É uma estrutura de dados que combina estado (campos) e ações (métodos)
   - Objetos: São instancias de uma classe
 ## Exemplo de Classes e Objetos

 ``` public class Ponto
 {
     public int x, y;
     public Ponto (int x, int y)
     {
       this.x = x;
       this.y = y;
     } 
 } 
```
